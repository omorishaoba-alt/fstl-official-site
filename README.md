# FSTL Official Site - Pilot App

This repository contains the **FSTL OSHOBE Pilot App**, fully self-contained in a single `index.html` file.

## Features

- Fully modular pilot app
- Agents can add dynamic modules
- Fully self-contained (no external CSS or JS)

## Deployment on GitHub Pages

1. Push this repository to GitHub.  
2. Go to **Settings → Pages → Source → main branch / root** → Save.  
3. Wait 2–3 minutes.
